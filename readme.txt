make sure you have simso and simsogui installed if it is not already.
pip install simso
pip install simsogui

navigate to C:\Users\<username>\AppData\Roaming\Python\Python38\site-packages\simso
or wherever simso is installed on your computer

Copy and paste the BJF.py and NOPSACK.py to the 'schedulers' folder in SimSo

Run the main.py in terminal or VScode.

in the simso gui window that just opened. Load the xml file for the scheduler you want to run.

Press run and check results and Gantt charts.
Done :)
